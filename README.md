## Description
pyrua is a simple Python module for generating random User-Agent strings for web scraping purposes.

## Installation
You can install pyrua using pip:
```
pip install pyrua
```

## Usage
```python
from pyrua import get_rua

user_agent = get_rua()
print(user_agent)
```
